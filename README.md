# EasyEDA-Connected-CircularLED
A DIY project to create a professional-looking smart LED lamp using EasyEDA for PCB design and JLCPCB for fabrication. This project demonstrates how to design, fabricate, and assemble a circular LED lamp with a D1 Mini microcontroller and WS2812B RGB LEDs.
